To install to the robot:

Libraries install:
- FTP to your robot.
- Make a new directory at the root called "lib".
- Make a new directory inside "lib" called "python3.1".
- Copy all files and directories from Python/Lib to the newly created
  /lib/python3.1.

Boot install.  The C code runs py/boot.py; a standard version is provided
in the py/ directory.
- FTP to your robot.
- Make a new directory at the root called "py".
- Copy py/boot.py from the source tree to the robot /py directory.

Robot install.  boot.py imports robot.py to do its work.  A sample template
is provided in samples/template.
- FTP to your robot.
- Copy robot.py to the robot /py directory.
